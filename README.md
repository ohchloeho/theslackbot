# Documentation for THESLACKBOT
Create App on Slack Workspace Settings:

```manifest.json
{
    "display_information": {
        "name": "The Bot"
    },
    "settings": {
        "org_deploy_enabled": false,
        "socket_mode_enabled": false,
        "is_hosted": false,
        "token_rotation_enabled": false
    }
}
```
